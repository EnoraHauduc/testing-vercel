# Maths BSc Welcome

A Next.js welcome page for incoming Mathematics BSc students: what to expect,
core first-year topics, study tips, and resources.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploying to Vercel

This project is ready to deploy as-is — import the repository into
[Vercel](https://vercel.com/new) and it will be detected automatically as a
Next.js app. No environment variables are required.
